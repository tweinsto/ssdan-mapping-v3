Sources & Methodology

Data Sources

  • American Community Survey (ACS) 5-Year Estimates, 2023: Used for most county and state-level demographic, economic, and housing variables.
  • Decennial Census (2000): Used for comparison values, especially in population growth and change over time visualizations.

Variable Documentation

Data was retrieved using variable codes from the Census API. Key variables include:

  • B20004_001 – Median Earnings
  • B15003 – Educational Attainment
  • B17001 – Poverty Status
  • B01001 – Age Distribution
  • B03002 – Race and Ethnicity

Data Access and Tools

  • Data was accessed using the tidycensus package in R, which interfaces with the Census Bureau API.
  • Data processing and cleaning were done using dplyr, tidyr, and purrr.
  • Geographic data (county/state shapes) were handled using sf.
  • Maps were rendered using leaflet for interactivity and saved as static HTML.
  • Charts were generated using ggplot2 and plotly.

Map and Visualization Notes

  • Color scales were generated with RColorBrewer palettes, adjusted to match topic range.
  • Some states have customized color breaks for clarity, especially in small-range variables.
  • Static HTML maps were exported for performance and integrated into the site structure.

Attribution

  • Data provided by the U.S. Census Bureau. While data is public, visualizations and interpretations are original to SSDAN.
  • Mapping and analysis tools are built in R using open-source packages.
